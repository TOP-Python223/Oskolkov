Стандартная библиотека: datetime, itertools, functools, operator, time, pathlib. Встроенные функции.

  ==========  1  ==========

Составьте скрипт для формирования расписания занятий годовых курсов с учётом выходных дней.

Для тестирования используйте наше расписание:

         старт: 23.04.22
         всего: 140 пар (две в день)
    дни недели: Сб, Вс
      выходные: 08.05.22
                01.08.22–14.08.22
                31.12.22–08.01.23

Необходимо выводить дату каждой пары на новой строке в формате ДД/ММ/ГГГГ



  ==========  2  ==========

Используя бесконечный счётчик из модуля itertools и прочие последние знания напишите в одну строку компактное формирование словаря с нумерованным английским алфавитом.



  ==========  3  ==========

Напишите декоратор повторного вызова.

В случае, если вызов оборачиваемой функции прекращается с исключением, функция должна быть вызвана повторно через 5 секунд.

Используйте функцию sleep из модуля time.



  =========================

Код каждой решённой задачи помещается в текущий каталог репозитория в виде отдельного файла с именем, соответствующем номеру задачи.

Функции обязательно документируются.
Параметры функций обязательно аннотируются.

Все файлы выполняются с помощью интерпретатора.
Текстовый вывод в стандартный поток stdout копируется.
Копия вывода в виде комментария помещается в конец файла с кодом задачи.

В сервис Журнал отправляется сообщение о выполнении задания со ссылкой на репозиторий.

Преподаватель пишет комментарии и правки в коде задач и обновляет файлы в удалённом репозитории.
Проверяйте обновления в своём репозитории каждый день.
